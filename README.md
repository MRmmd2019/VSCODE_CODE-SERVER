<h2 align="center">🧠 درباره Termux | About Termux</h2>

<div align="center">

📱 ترموکس (termux) یه امولاتور ترمینال و محیط لینوکسی برای اندرویده که بدون روت اجرا می‌شه <br>
📦 می‌تونی با `pkg` یا `apt` هر پکیجی که خواستی نصب کنی <br>
🔗 اطلاعات بیشتر توی [termux.com](https://termux.com) <br><br>

📱 Termux is a terminal emulator and Linux environment for Android — no root required! <br>
📦 You can install any package using `pkg` or `apt` <br>
🔗 More info at [termux.com](https://termux.com)

</div>

### 🔧 مرحله نصب | Installation Steps

```bash
# 📦 برو به ترموکس و اینا رو اجرا کن
# Run these commands in Termux
pkg upgrade && pkg install -y git
```

```bash
# 📥 ریپو رو کلون کن | Clone the repo
git clone --depth=1 https://github.com/MRmmd2019/VSCODE_CODE-SERVER
```

```bash
# ▶️ اجرا و نصب | Run the installer
cd VSCODE_CODE-SERVER
chmod +x setup.sh
./setup.sh
```

> 💡 اگر نصب با مشکل مواجه شد، دوباره اجرای setup.sh رو امتحان کن!  
> 💡 If setup fails, try running `setup.sh` again.
